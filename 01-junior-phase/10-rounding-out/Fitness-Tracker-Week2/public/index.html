Welcome to the Fitness Tracker API

Check out the following API routes:

  • GET all workouts
  • GET all exercises

  • GET single workout by id
  • GET single exercise by id

  • POST a new workout with this request body (JSON): 
      {
    "name": "Arm Day",
    "date": "2019-03-05T15:39:20.199Z"
  }
  • POST a new exercise with this request body (JSON): 
      {
    "name": "Swimming",
    "description": "Swim 10 laps freestyle",
    "completed": false
  }

  • DELETE workout by id
  • DELETE exercise by id

Postman is a great way to play around with the POST and DELETE routes
